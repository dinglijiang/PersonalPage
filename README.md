# PersonalPage

This repository hosts Dingli Jiang's personal academic homepage on GitHub Pages.

Live site:

https://dinglijiang.github.io/PersonalPage/

## Structure

- `index.html`: homepage content
- `personal-assets/styles.css`: visual styles and responsive layout
- `personal-assets/script.js`: navigation, scroll progress, and reveal animations
- `personal-assets/research-flow.png`: research visual used in the hero section
- `.nojekyll`: tells GitHub Pages to publish the static files directly

## Editing

The site is a plain static page. No Jekyll, Ruby, Node.js, Docker, or build step is required.

After editing, commit and push to the `master` branch. GitHub Pages publishes from the repository root.
